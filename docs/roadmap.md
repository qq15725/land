# Roadmap

## 阶段划分

### Phase 0 · 基础骨架
> 目标：能跑起来的最小世界，玩家能动、能采集、能捡东西

- [ ] 项目结构搭建（目录、Autoload注册）
- [ ] 玩家移动（8方向 + 动画状态机）
- [ ] 摄像机跟随
- [ ] Y-sort 层级排序
- [ ] 基础世界生成（单Biome、平铺地形）
- [ ] 可采集资源节点（树/石，带交互提示）
- [ ] 物品数据层（ItemResource 基类）
- [ ] 背包系统（数据层，无UI）
- [ ] 掉落物 + 拾取

---

### Phase 1 · 建造与合成
> 目标：能采集 → 合成 → 建造基地

- [ ] 合成系统（配方数据 + 手动合成逻辑）
- [ ] 工作台建筑（解锁更多配方）
- [ ] 建造系统（选蓝图 → 预览 → 放置 → 消耗材料）
- [ ] 建筑耐久 + 修复
- [ ] 仓库建筑（扩展存储）
- [ ] 背包 UI（格子式，拖拽）
- [ ] 合成面板 UI
- [ ] HUD（生命值、当前工具）

---

### Phase 2 · 农场
> 目标：能种菜、能养殖、产出形成循环

- [ ] 耕地系统（锄地 → 播种 → 浇水 → 收获）
- [ ] 作物生长周期 + 数据驱动
- [ ] 四季影响可种品种
- [ ] 围栏建筑
- [ ] 动物 AI（游荡、被圈养）
- [ ] 喂食 + 繁殖逻辑
- [ ] 动物产出采集（蛋/奶/皮毛）
- [ ] 烹饪系统（食材 → 食物 → 恢复生命）

---

### Phase 3 · 世界与怪物
> 目标：世界有内容，有探索价值，有威胁感

- [ ] 多 Biome 世界生成
- [ ] Chunk 按需加载/卸载
- [ ] 昼夜循环 + 光照变化
- [ ] 四季系统
- [ ] 怪物游荡 AI（状态机）
- [ ] 怪物攻击行为（夜晚/被激怒）
- [ ] 玩家战斗（近战攻击、击退）
- [ ] 死亡 + 复活流程
- [ ] 怪物掉落表

---

### Phase 4 · 交易
> 目标：产出有出口，稀有内容有获取途径

- [ ] 贸易站建筑
- [ ] 商人 NPC（周期性来访）
- [ ] 商人交易表（数据驱动）
- [ ] 交易面板 UI
- [ ] 不同商人携带不同商品

---

### Phase 5 · 打磨 & 体验
> 目标：可玩的完整 Demo

- [ ] 存档系统（序列化 + 多存档槽）
- [ ] 主菜单 + 存档选择界面
- [ ] 暂停菜单
- [ ] 音效 + 背景音乐
- [ ] 粒子效果（采集、受伤、收获）
- [ ] 游戏内教程/新手引导
