# 地砖 Autotile Atlas 生成提示词

## 输入文件

| 文件 | 用途 |
|------|------|
| `docs/references/terrain.png` | 风格参考图（等角 voxel 像素小岛，只取顶面纹理语言） |
| `docs/references/tile_template.png` | 结构模版图（512×384 px，8列×6行，47格 tile，四种地形共用） |

---

## 模版说明

`tile_template.png` 用两种颜色编码了 47 格 blob autotile 的边缘连通结构：

- **浅灰** — 该方向与同地形相邻，纹理无缝延伸到像素边缘
- **深灰（边缘带）** — 该方向暴露，需绘制 4–6px 过渡收边；角落的深灰方块表示凹角缺口

生成时**严格保持模版的结构**，只替换颜色和纹理。

---

## 通用生成指令

```
参考图1（风格）：docs/references/terrain.png
参考图2（结构模版）：docs/references/tile_template.png

以参考图2为精确结构模版，输出 512×384 px 的地形 atlas：
- 完全保持每格 tile 的边缘结构：有深色 fringe 带的边保留收边，无 fringe 的边纹理延伸到像素边缘
- 完全保持角落 notch 缺口的位置和形状
- 网格对齐与模版完全一致，输出无额外边距或留白

像素风格，硬边缘，无抗锯齿，相邻 tile 无缝拼接。
从参考图1提取纹理语言，忽略其侧面、阴影厚度、水体及场景物件，只取顶面平铺纹理风格。

[粘贴下方对应地形的纹理描述]
```

## 各地形纹理描述

**草地**
```
内部：明亮中绿 ~#4A8A28，散布深绿像素簇（约30%覆盖），偶有亮黄绿高光像素，轻微有机噪点
fringe：暖棕 ~#7A5030，2–3px 草色收窄后接 2px 暖棕；边缘有 1–2px 不规则草尖像素向外突出
```

**小路**
```
内部：暖米色 ~#C8A060，有机砂砾像素簇（棕褐 ~#A07840），散布 1–2px 深色砾石 ~#705030
fringe：略浅干沙 ~#D4B070，2px 轻微提亮，近矩形收边，少量有机变化
```

**耕地**
```
内部：深巧克力棕 ~#4A2810，严格水平犁沟线（1px 隆起 #5E3418 + 3px 凹槽 #381A08，从 y=0 起重复，所有 tile 相位锁定）
fringe：最深棕 #381A08，犁沟到边缘干净终止，1px 暗边，近直线（人工感，少有机变化）
```

**石地**
```
内部：中灰 ~#787878，不规则裂缝线（非规则网格），浅灰方形高光块 ~#989898，深灰裂缝 ~#585858
fringe：深灰 ~#585858，2px 加深收边，裂缝线延伸至边缘后终止，略有锯齿像素变化（碎石质感）
```

## 输出规格

| 参数 | 值 |
|------|-----|
| 尺寸 | 512×384 px |
| 网格 | 8列 × 6行，每格 64×64 px |
| 最后一格 | 留空（透明或黑色） |
| 保存路径 | `assets/sprites/environment/terrain_[地形].png` |
